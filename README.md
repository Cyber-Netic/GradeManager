# 🎓 Java Grade Manager

A **simple student grade manager** built using **Java**. Grades are saved in a file for persistence.

## 🚀 Features
- ✅ **Add student grades**
- 📋 **View all grades**
- ✏️ **Update student grades**
- 🗑️ **Delete grades**
- 💾 **Stores grades in a text file**

## 📌 Technologies Used
- **Java** (Programming Language)
- **File I/O** (Persistent Data Storage)
